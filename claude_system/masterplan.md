# masterplan.md

## Product
tDelta = chrome extension, self-binding accountability via persistent time awareness

## Core Idea
voluntary discomfort > reminders  
visual time pressure + stats = behavior change

## Target
students doing assignments + study sessions

## V1 Scope
- chrome extension only
- local storage only
- free
- no backend
- no AI
- minimal UI

## Core Features
1. persistent bar under tab bar
2. up to 3 active tasks
3. manual lock-in + manual complete
4. color escalation over time
5. domain/tab tracking
6. optional distraction blocking
7. 15-min retrospective summaries
8. local history + stats

## Non-Goals
- cloud sync
- calendar integration
- automation
- smart detection
- social features

## UX Rules
- bar always visible when locked
- no shame language
- neutral reporting
- no forced exits
- friction is intentional

## Metrics
- lock-in activation rate
- avg session length
- completion rate
- weekly retention
- % focused time

## Mitigations
- first lock-in = short (10 min)
- clear value demo on onboarding
- graduated blocking severity

## Roadmap
V1: core loop + stats  
V2: better visuals + presets  
V3: optional leaderboards (opt-in)

## Success Definition
students repeatedly choose discomfort
